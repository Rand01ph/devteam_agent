## 周报整理流程

用户会将团队成员的原始周报内容粘贴到 `## 待整理周报` 段落中，格式如下：

```
## 待整理周报

## 成员名1

**今日工作总结**

（成员1的工作内容...）

## 成员名2

**今日工作总结**

（成员2的工作内容...）
```

当用户说"整理周报"或类似指令时：

1. 确定日期范围（年、月、周数、起止日期）
2. 调用 `organize_weekly_report` 工具
3. 工具会自动：
   - 从 `## 待整理周报` 中读取原始内容
   - 识别所有 `## 成员名` 段落
   - 移除 `**今日工作总结**` 标题
   - 将内容整理到标准结构（Agent总结、个人总结、工作明细）
   - 清空 `## 待整理周报` 段落
4. 汇报整理结果，说明已整理的成员列表
